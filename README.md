# mlprod-ai-library
Collection of my favorite AI prompts, skills, markdown docs, etc

Some are ones I created personally, some are taken from various sources and modified for my personal tastes. Feel free to utilize and modify for your needs as well. 

If you come across a refinement that makes any of these skills better, drop me a PR on it. Always looking for improvements.

This library is dynamic and will grow and change as I learn better way to do things. Most of the items here are configured for use with Claude, or vscode agents.
